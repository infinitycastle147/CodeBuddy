# 🚀 CodeBuddy - Your AI-Powered Code Companion

**Transform the way you understand, explore, and collaborate with code**

CodeBuddy is an intelligent code analysis platform that bridges the gap between developers and their codebases through conversational AI, automated insights, and visual code exploration. Whether you're onboarding to a new project, conducting code reviews, or exploring complex architectures, CodeBuddy makes code comprehension effortless.

---

## ✨ Why CodeBuddy?

### 🤖 **Chat with Your Code**
- Have natural conversations about any codebase
- Get instant explanations of complex functions and algorithms  
- Ask questions like "How does authentication work?" or "Show me the API endpoints"
- Multi-agent AI system provides accurate, context-aware responses

### 📊 **Visualize Code Architecture**
- Automatically generate beautiful Mermaid diagrams from your code
- Create flowcharts, sequence diagrams, and class diagrams
- Perfect for documentation, presentations, and architectural reviews
- Export diagrams in multiple formats

### 🔍 **Intelligent Code Explorer**
- Browse codebases with AI-powered semantic search
- Find relevant code snippets using natural language queries
- Multi-language support: Python, JavaScript, TypeScript, Java, Go, C++, C#
- Vector-based similarity search finds related code patterns

### 🛡️ **Security-First Design**
- Enterprise-grade security with encrypted credentials
- Safe repository processing with automatic cleanup
- Input validation and prompt injection protection
- Secure GitHub integration for private repositories

## 🚀 Getting Started

### **Quick Setup**

1. **Clone the Repository**
   ```bash
   git clone <repository-url>
   cd CodeBuddy
   ```

2. **Start with Docker (Recommended)**
   ```bash
   # Start all services
   docker-compose up --build
   
   # Access the application
   # Frontend: http://localhost:3000
   # Backend API: http://localhost:8000
   ```

3. **Connect Your Repository**
   - Navigate to the setup page
   - Enter your GitHub repository URL
   - Optionally provide access token for private repos
   - Let CodeBuddy analyze your code

4. **Start Exploring**
   - Chat with your code using natural language
   - Generate diagrams from your codebase
   - Browse files with intelligent search
